# Exercise 1

Let $\alpha = a + bi$ where $a, b \in \mathbb{R}$.

Let $\beta = c + di$ where $c, d \in \mathbb{R}$.

Then,

```math
\begin{align*}
\alpha + \beta &= (a + bi) + (c + di) \\
&= a + bi + c + di \\
&= a + c + bi + di \\
&= (a + c) + (bi + di) \\
&= (a + c) + (b + d)i \\
&= (c + a) + (d + b)i \\
&= c + a + di + bi\\
&= c + di + a + bi\\
&= (c + di) + (a + bi)\\
&= \beta + \alpha
\end{align*}
```
